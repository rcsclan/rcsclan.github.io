INSTALACAO:

- O de_sampa nao eh um jogo completo, eh apenas um mapa feito para o Counter-Strike, que por sua vez eh uma modificacao do Half-Life. Portanto, voce primeiro precisa ter o jogo Half-Life (vendido em qualquer boa loja de software) instalado e atualizado. Depois disso, instale a ultima versao do Counter-Strike.
- Abra o arquivo de_sampa.zip com o Winzip.
- Descompacte (Extract) tudo para o diretorio C:\Sierra\Half-Life\cstrike\ (troque o C:\Sierra\Half-Life\ pelo diretorio onde esta instalado o seu Half-Life) Certifique-se de que a opcao Use folder names na janela de descompactacao do Winzip esteja ligada.
- Ache um servidor na internet e divirta-se!
- Se voce ainda nao tem o Half-Life ou o Counter-Strike instalado, nao sabe como conectar num servidor ou nao tem ideia do que estou falando, veja o Guia BRGames.

JOGANDO COM BOTS:

- Para jogar com bots, voce primeiro precisa instalar o PODBot.
- Este mapa ja vem com waypoints ("roteiro" do mapa para os bots) necessarios para o PODBot.
- Abra o Counte-Strike e inicie um Jogo em LAN (Play CS -> Play On LAN -> Create).
- Selecione o mapa de_sampa e o numero de jogadores (voce+bots).
- Quando o mapa entrar, os bots deverao ser adicionados automaticamente.
- Use a tecla "=" (sinal de igual) para acessar o menu de controle dos bots.

MAIORES INFORMACOES:

- Para aprender a mapear, leia os meus tutoriais no Mondo Bizarro.
- Para tirar duvidas sobre edicao de mapas, por favor use os forums Made In Brazil.
- Para ficar por dentro do mundo Counter-Strike, visite regularmente o CS-NEWS e a Hardmob.

INSTALLATION:

- Open the file de_sampa.zip with Winzip.
- Extract everything to the directory C:\Sierra\Half-Life\cstrike\ (change C:\Sierra\Half-Life\ to the directory where you have Half-Life installed). The option Use folder names must be ON.
- For an english breafing screen, after the installation go to the directory Half-Life\cstrike\maps\, remove the file de_sampa.txt and rename file de_sampa_en.txt to de_sampa.txt
- Find a server and have fun!
